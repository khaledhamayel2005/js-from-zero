# JavaScript from Zero

This repository contains a comprehensive JavaScript learning journey, organized into structured folders for progressive learning.

## Project Structure

### basics/
Fundamental JavaScript concepts and syntax:
- **01-data-types/**: Introduction to primitive data types (string, number, boolean, etc.)
- **02-variables/**: Variable declaration with var, let, and const
- **03-operators/**: Arithmetic and assignment operators
- **04-type-coercion/**: Type conversion and coercion examples
- **05-string-methods/**: String manipulation methods
- **06-number-methods/**: Number methods and operations

### intermediate/
Intermediate JavaScript concepts:
- **01-functions/**: Function declarations, expressions, and arrow functions
- **02-arrays/**: Array creation, manipulation, and methods
- **03-objects/**: Object creation, properties, and methods
- **04-loops-conditions/**: Loops (for, while) and conditional statements
- **05-higher-order-functions/**: Map, filter, reduce, and other functional methods

### advanced/
Advanced JavaScript topics:
- **01-dom/**: Document Object Model manipulation and events
- **02-scope/**: Variable scope and closure concepts

### projects/
Complete JavaScript applications:
- **01-todo-app/**: Task management app with local storage
- **02-calculator/**: Calculator with keyboard support
- **03-weather-app/**: Weather application with API integration

### notes/
Learning notes and documentation.

### scripts/
Utility scripts for repository management.
